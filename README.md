# php-dockerfiles
PHP base dockerfile for php 8.0.15 based on php:8.0.15-apache (debian)

Includes a number of packages, including:
- nodejs
- git
- vim
- net-tools
- silversearcher-ag
- bat
- PECL
- MSSQL && MSSQL PDO drivers (Debian 10)
- mysql PDO drivers
- php intl, iconv extensions
- xdebug
- composer
